# RAKLIFY Portfolio Website

## Overview
RAKLIFY is a premium, modern, and responsive portfolio website for a digital marketing & software development agency. The site showcases services, projects, a blog, and contact information with a sleek dark‑mode aesthetic, glass‑morphism effects, and smooth micro‑animations.

## Features
- **Responsive design** – works on mobile, tablet, and desktop.
- **Dynamic sections** – Home, Services, Portfolio, Blog, Contact.
- **Animated hero & tech‑stack marquee**.
- **Pure vanilla stack** – HTML5, CSS3, JavaScript (no frameworks).
- **SEO‑friendly meta tags** and semantic HTML.
- **Easy to customize** – all styles are in `style.css` and scripts in `script.js`.

## Tech Stack
- **HTML5** – semantic markup.
- **CSS3** – custom design system, gradients, glass‑morphism, animations.
- **JavaScript (ES6+)** – interactivity and DOM manipulation.

## Getting Started
1. Clone the repository (or download the ZIP).
2. Open `index.html` in any modern browser.
3. No build step or server is required – it runs completely client‑side.

```bash
# Example (if you have Git installed)
git clone https://github.com/AJITHKANNAN03/raklify_profile.git
cd raklify_profile
# Open the site
start index.html   # Windows
# or double‑click the file in Explorer
```

## Customisation
- Edit `style.css` to change colors, fonts, or layout.
- Modify `script.js` for additional interactive behaviour.
- Update the HTML files (`index.html`, `about.html`, etc.) to reflect your own content.
